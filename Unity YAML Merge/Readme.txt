Настройку кастомного драйвера слияния Unity YAML Merge можно выполнить одним из двух способов:

1)отредактировать код в файле config.txt изменив путь к UnityYAMLMerge.exe на правильный и 
добавить этот код в файл config находящийся в папке .git (в корне репозитория)

2)Создать в папке Assets (в корне репозитория) скрипт с кодом из файла auto.txt который 
выполнит настройку автоматически при первом открытии проекта
